## Deployment

This site is fully static and ready to be deployed via GitHub Pages.

### To Deploy

1. Push to a GitHub repo
2. Go to Settings > Pages
3. Select branch: `main`, folder: `/ (root)`
4. If using a custom domain, add `CNAME` file
